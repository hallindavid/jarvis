a terminal based todo app.

jarvis was created by, and is maintained by [Savio](https://github.com/lotharthesavior), and [Dave](https://github.com/davidhallin)

## commands

// Search

```
j ls query?
```

// List

```shell
j ls
```

// Create

```shell
j mk  
```

// Delete

```shell
j rm
```

// Push

```shell
j push
```

// Pull

```shell
j pull
```
